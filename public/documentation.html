Documentation

This API will list directors from movies

Post

/users - Allows you to create Users
/users/:Username/Movies/:MovieID - Allows you to add favorite movies for users

GET

/movies - Lists all movies on the movie_api
/movies/:Title - Gives information about a specific movies
/movies/genre/:name Lists information about a specific genre
/movies/Director/:Name - Lists Information about the Director
/users - Lists Users and their favorite movies
/users/:Username - looks up specific users

Put

Allows you to update information about directors
/users/:Username - Allows you to update infomration about users

DELETE

/users/:Username/Movies/:MovieID - Allows you to delete a movie from a user's favorite list
/users/:Username - allows a user to delete his account
